Реализовано B-дерево
Операции оптимизированы, чтобы минимизировать обращения к диску 
для чтения и записи.
В папке input - тесты, в папке output - ответы на эти тесты.

Для тестирования программы необходимо задать 4 параметра командной строки:
BTreeStringValue.exe <t> <folder> <inputPath> <outputPath>
t - параметр B-дерева
folder - путь к папке, в которую сохранять файлы с узлами дерева
inputPath - путь к файлу с тестом
outputPath - путь к файлу, в который записать ответ.
Пример:
BTreeStringValue.exe 5 folder input/test6.txt output/answer6.txt

В файле с тестом можно использовать команды:
1)insert <key> <value>
Например, insert 1 asdfgY13
Вставить в дерево новый ключ.
В ответ на это программа вернет true, если ключ успешно вставлен, и false,
если такой ключ уже есть.
2)find <key>
Например, find 5
Найти значение по ключу.
В ответ программа вернет значение, если оно есть, и null, если такого ключа
нет.
3)delete <key>
Например, delete 5
Удалить значение по ключу.
Из дерева будет удалено значение с данным ключом. В ответ программа вернет
удалённое значение, или null, если такого значения не было.
4)[] <i>
Например, [] 0
Найти значение по индексу.
В ответ программа вернёт значение, соответствующее i-ому по возрастанию 
ключу. Если в дереве <= i ключей, то будет выведено null.
Команды разделяются началом новой строки.
